Storm AT  Automata Theory Calculator

To close this help window, click the X at its top-right corner.

Input Area

The first text area, enter valid arithmetic operations ending with an equal sign (=) then press Enter.

Valid characters: Numbers (0-9), Arithmetic Operators (+, -, *, /), end-of-line equal sign (=).

Invalid inputs: Non-valid characters, no operator between numbers, no numbers between operator, equal sign anywhere but end of line.

Message Area

Between the two text areas, displays any error the input or its operations may have.

  • Invalid Expression: An invalid character or input was given
  • Arithmetic: An invalid arithmetic operation occurred like division by zero.
  • Operator: Operators cannot be stacked unless to denote negative numbers once.
  • Overflow: A number has exceeded the max digit limit, which is 12.

Operations Area

The lower text area, shows the operations as the calculator executes them from the input area.
